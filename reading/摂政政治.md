# 朝賀(ちょうが)

元日に、諸臣が朝廷に参集して天皇に新年のおよろこびを申し上げること。

平安時代前半はやってたけど

後半から天皇サイドの力がおちて、

藤原くんがやりたくないといってやらないことになって略式の小朝拝になった。

```
藤原「やめろ」
天皇「はい」

~905年~

醍醐天皇「やれ」
藤原「...はい」

~14 年後~

藤原「やっぱやめろ」
天皇「はい」
```
