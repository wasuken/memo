# ガロン=バルッオツォ=ゾィオイス=ティアとは

自分がつくった宗教。

原型の言葉は「がんばるぞい」

これは

生活の中で**考えるまでもなく**、やるべきこと、まもるべきことをまとめたもの。

それをいつまでもまもれなかったり意思がよわく、継続ができないので宗教というつよいものとして定義することにした。

作成過程もかいているが、みてのとおりかなりふざけている。
